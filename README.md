# Сортировка вставками

Программа сортирует массив целых чисел по возрастанию методом сортировки вставками.

При запуске предлагается выбрать способ ввода данных:
1 - Ручной ввод
2 - Генерация случайных уникальных чисел в заданном диапазоне
3 - Чтение массива из файла result.txt

Результат сортировки выводится на экран и сохраняется в файл result.txt.
